# Phase 4 Context

The current Web Audio graph has one master gain and one reverb return directly into it. That makes truthful category mute impossible. Phase 4 introduces normalized persistent settings, semantic buses, category-owned reverb returns, and native keyboard-operable controls on title and pause screens.

Controls: master, ambience, music, effects, voice, jumpscare intensity, and master mute. Slider interaction must never start play or request pointer lock.
