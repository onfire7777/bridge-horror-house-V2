# BridgeMind parody assets

These assets were prepared for the BridgeMind Horror House parody from the public
BridgeMind livestream [Vibe Coding With GPT 5.6](https://www.youtube.com/watch?v=MkeyOv0P7TY),
published July 9, 2026.

- `matt-billboard.png`: identity-preserving cutout derived from the stream thumbnail,
  with the intentionally crude white sticker border requested for the game.
- `bridgemind-techno-loop.wav`: short excerpt from `00:00.25–00:18.90`.
- `voice/lets-go-chat.wav`: short excerpt from `00:19.18–00:21.16`.
- `voice/lets-go-loop.wav`: short excerpt from `00:27.30–00:28.98`.
- `voice/token-efficient.wav`: short excerpt from `06:29.10–06:33.64`.
- `voice/what-is-that.wav`: short excerpt from `19:59.10–20:01.35`.
